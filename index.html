Work

SpeedPass Micro Site

Speedpass+ is an app by Exxon Mobile that was released in limited locations in the summer of 2013. Tribal Worldwide was assigned the task of developing a promotional microsite for the app. We used twitter bootstrap to create a responsive one page experience. I developed the majority of the html and css myself as I was the main developer on the project with oversight from the tech lead.

Cano Retouching Site

This was a site I built using galleria for retoucher Daniel Cano. Customizing the look of the plugin and giving it an original feel was the main challenge of the project. I also added some interactivity with javascript allowing the user to view the pre-retouched photo upon rollover certain images.

Matt Pizano Site

Matt Pizzano was a director who wanted a custom site design to showcase his video work. He needed origin graphics for the design that followed a surrealistic art direction and have the layout work in wordpress so he could manage the videos. I was able to provide both the graphics and development which made this a fun project for me personally. I worked closely with Matt to narrow down the look of the site. The main focus was a drive in movie setting taking place on a city block. I also created special avatar designs for the about me section.
